3. Technical Implementation

For Dahuss, the state management required a solid architecture to handle complex administrative rules seamlessly without compromising performance.

Role-Based Access Control (RBAC)

We implemented dynamic rendering based on the user's role to securely guard the administrative routes directly on the client side, paired with robust backend checks.

// ProtectedRoute.js
const ProtectedRoute = ({ children, allowedRoles }) => {
  const { user, loading } = useAuth();

  if (loading) return <LoadingSpinner />;
  
  if (!user || !allowedRoles.includes(user.role)) {
    return <Navigate to="/login" replace />;
  }

  return children;
};
