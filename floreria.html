Business value

The site positions Florería El Nuevo Pensamiento online in Valparaíso—presenting floral products and in-person service clearly, driving direct WhatsApp inquiries, map visits, and visual exploration without a heavy app or complex backend.

The problem

The florist needed a clear, fast, trustworthy digital presence for products, hours, location, and contact channels—without relying on a heavy application or custom server. The goal was to turn a traditionally in-person and social-media-driven offer into an organized, indexable, mobile-friendly web experience.

The solution delivers:

  • Visual catalog with collapsible sections and image modals
  • Direct WhatsApp, phone, and Google Maps integration
  • Institutional video, FAQ, and Instagram feed
  • Local SEO with structured data for search engines

Development methodology

I delivered the full site from structure to production deployment. Here is how it was built:

  1. 1 — Static architecture

    A lean static stack with organized assets by product category—no custom server, minimal complexity, and delivery through a global CDN.

  2. 2 — Semantic content structure

    Clear page blocks: hero, local value proposition, video, services, catalog, contact, FAQ, Instagram, and footer—with semantic HTML, local SEO copy, and direct links to WhatsApp, phone, Maps, and social channels.

  3. 3 — Responsive visual design

    CSS variables, mobile-first layout, product grids, service cards, horizontal carousels, image modal, and floating WhatsApp button—prioritizing fast reading and a floral brand aesthetic.

  4. 4 — Vanilla JavaScript interactions

    Smooth scrolling, active section navigation, image modals, collapsible blocks, and carousels—lightweight logic without frameworks for the core experience.

  5. 5 — Technical SEO & structured data

    Meta descriptions, canonical URLs, social previews, geolocation hints, and structured data for local business and FAQs—plus sitemap and discovery files for search engines.

  6. 6 — Deployment & hardening

    Static hosting with clean URLs, image caching, and production security settings that limit risky scripts and embeds—without publishing how the site is configured.

Technical challenges solved

Maximum performance and discoverability for a local business—minimal infrastructure, consistent data, and safe third-party embeds.

Zero backend load

Fully static delivery via CDN—no server rendering or dynamic queries. Lightweight client optimizations, lazy images, and carousels that activate only when their section is visible.

Content & SEO consistency

Aligned visible content with structured data: business name, address, phone, hours, products, and location—so visitors and search engines see the same information.

Security without custom auth

No private APIs or user accounts—the site uses production security settings and official embeds for maps, video, and social content, without handling sensitive credentials in the project itself.
