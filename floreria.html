1. Project Overview and Objectives

Floreria El Nuevo Pensamiento is an elegant e-commerce platform built to seamlessly connect customers with beautiful floral arrangements. The goal was to provide a fresh, inviting digital storefront with a smooth purchasing flow.

Key Objectives:

  • Implement an intuitive product catalog and cart system.
  • Ensure a fully responsive and mobile-friendly shopping experience.

3. Technical Implementation

The application leverages a component-based architecture for managing dynamic product listings and shopping cart state efficiently.

State Management

Global state management was implemented to ensure the shopping cart remains synchronized across the entire platform.

// CartContext.js
export const useCart = () => {
  const context = useContext(CartContext);
  if (!context) {
    throw new Error('useCart must be used within a CartProvider');
  }
  return context;
};
